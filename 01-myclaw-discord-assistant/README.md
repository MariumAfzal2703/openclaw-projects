# 🐾 MyClaw – Discord AI Assistant

AI-powered Discord assistant built using OpenClaw.

---

## ⚡ Overview

MyClaw is a real-time Discord AI assistant that uses OpenClaw + LLM integration to respond intelligently inside Discord servers.

This project is my first practical AI agent implementation.

---

## 🚀 Features

- 💬 Real-time Discord responses  
- 🧠 AI-powered conversations  
- 🔗 OpenClaw integration  
- ⚡ Live tested bot workflow  

---

## 🛠️ Tech Stack

- OpenClaw  
- Node.js  
- Discord API  
- Google Gemini API
